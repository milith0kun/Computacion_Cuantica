# ⚛️ Computación Cuántica - UNSAAC

Bienvenido al repositorio de los proyectos y trabajos desarrollados para el curso de **Computación Cuántica** en la **Universidad Nacional de San Antonio Abad del Cusco (UNSAAC)**, Departamento Académico de Informática.

## 📂 Estructura del Repositorio

Actualmente, este repositorio contiene el trabajo principal del semestre:

- 📁 [**Proyecto Semestral**](./Proyecto%20Semestral/): Contiene la investigación e implementación de un modelo de **Aprendizaje Automático Cuántico (Quantum Machine Learning)** utilizando Qiskit. Este proyecto se encarga de comparar el rendimiento de clasificadores cuánticos (como VQC y QSVC) frente a modelos clásicos (SVM, MLP, KNN) utilizando el dataset Iris. Dentro de este directorio encontrarás el código fuente en Python, los notebooks de Jupyter de los experimentos, y los archivos LaTeX del informe final.
